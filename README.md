# wrs-followup

Projects followup details used for Web Rocket Services.

- Project summary
- Proposal
- List projects
- Meeting followup
- comments

## tech stack:

- Vite
- React
- Mui
- Typescript
- Jsx
- Vitest
- Redux
- Firebase

## Project Setup

```sh
git fork
npm install
```

Rename the .env-template to .env and configure your own values.

```sh
npm run dev
```
